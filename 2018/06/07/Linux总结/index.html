Linux总结

1. Linux目录

目录 功能 记忆
/bin 常用系统程序目录 binary
/boot 开技设定目录 boot
/dev 系统设备装置文件目录 device
/etc 系统配置文件，尤其是passwd，shadow文件 Edit Text Configuration
/home 系统使用者的目录
/lib Linux执行或编译程序函数库的目录 Libary
/mnt 软驱和光驱接入挂载的地方
/proc 系统核心与程序的一些信息
/tmp 临时文件目录 Temporary 临时的
/sbin 系统管理员常用指令集
/usr /usr/sbin 系统管理员常用指令集
/usr/bin 一般执行文件摆放的地方
/var 摆放系统日志文件
